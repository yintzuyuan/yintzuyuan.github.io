上個月決定測試看看 Claude Code 的 Vibe Coding 極限，毅然決然地訂閱了 Max 版進行九宮格預覽的重構和強化工作。
前兩天訂閱到期，這項工作也告一段落，成果我認為相當滿意，雖然我原本還想用這段時間再寫一點其他工具的。

老實說作為一個免費的外掛工具，九宮格預覽已經到達了我可以接受的極限，功能再多下去我就準備要收費了。
不過不用擔心，這款工具我會讓它永久免費下去。當然，作為我滿足自己的日常做字需求和程式碼練功的工具，它有時候會有一些嘗試性或是不太成熟的功能，但我會讓它不斷進化下去，並將從這段路程中學到的經驗和心得，用在其他更成熟的工具開發旅程上。

## v3.3.2 主要更新

這次更新背後，其實是我個人進行的一項小實驗 - Vibe Coding with Claude Code

上個月，我決定測試看看 AI 協作開發的極限，所以訂閱了 Claude Max 版，目標是將「九宮格預覽」進行一次徹底的重構和功能強化。從模組化架構、多國語言系統到核心的重繪邏輯，整個過程深度依賴 AI 進行 TDD 測試驅動開發和程式碼優化。

前兩天訂閱到期，這項工作也剛好告一段落。成果我相當滿意，整個外掛的穩定性和擴充性都提升到了一個新的層次。

### 更強大的預覽核心

除了零延遲的即時重繪，現在可以直接預覽備份圖層 (Backup Layers)，方便你隨時比較細微修改。同時，更與 Light Table 外掛深度整合，在九宮格中就能進行版本比對。

### 更智慧的 UI/UX

新增了無效字符的視覺化提示，輸入錯誤時能一目了然。也再次微調了 UI 佈局，讓內容的展示更清晰、操作更直覺。

## 關於這個工具的理念與未來

老實說，作為一個免費外掛，九宮格預覽的功能已經到達我個人設定的穩定階段了。

我承諾這款工具會永久免費，並會持續進行必要的維護。但可以預見的是，未來應該不會再有重大的新功能或架構重構被加入。我會將主要的開發心力，投入到探索和打造其他全新的字型工具上。

對我來說，九宮格預覽是一個很棒的練功專案，而從中學到的經驗，將會是開發下一個更成熟工具的基石。

最後，感謝所有使用和提供回饋的設計師們，你們的意見是這個專案持續優化的動力。
